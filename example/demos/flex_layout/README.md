# Flex Layout Demo

Demonstrates flexible layout with row and column directions.

## Basis Types

- `Px n` - Fixed pixel width
- `Percent p` - Percentage of available space
- `Ratio r` - Proportional share
- `Fill` - Take remaining space

## Features

- Gap spacing between children
- Padding around container
- Center alignment
- Space-between justification
- Resize to see responsive behavior
