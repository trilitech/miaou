# Miaou Invaders

A mini arcade shooter demonstrating Miaou rendering, animation,
timing, and game-loop capabilities.

## Features Used

- **Canvas** — cell-level game field rendering
- **Canvas_widget** — auto-sizing layout integration
- **Animation** — explosion effects with easing
- **Clock** — delta-time for frame-independent movement
- **Timer** — periodic alien movement and shooting
- **Powerups** — falling colorized `$` bonuses with upgradeable shot styles
- **Ship evolution** — ship form grows and changes as power stacks
- **Rare recovery** — occasional extra-life drops from combat/powerups
- **Procedural levels** — progressively harder waves with a boss every 5 levels
- **Screen shake** — impact feedback when the player is hit
- **Theme rotation** — readable background palettes that rotate each level
- **Title screen** — animated starfield intro

## Controls

- **Enter**: Start game from title screen
- **d**: Start AI demo mode from title screen
- **Left/Right** arrows or **h/l**: Move ship
- **Space**: Fire
- **p**: Pause/Resume
- **r**: Restart game
- **t**: Open this tutorial
- **Esc**: Return to launcher
