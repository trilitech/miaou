# Toast Notifications Demo

Displays toast notifications with different severity levels.

## Keys

- 1 - Show info toast
- 2 - Show success toast
- 3 - Show warning toast
- 4 - Show error toast
- b - Push to flash bus
- d - Dismiss oldest toast
- p - Cycle position
- Esc - Return to launcher

## Toast Severities

- Info, Success, Warn, Error
