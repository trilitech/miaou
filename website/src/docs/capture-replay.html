Course 5

Capture real terminal sessions.

Miaou includes capture and replay helpers for debugging, sharing demos, and creating documentation media from real terminal sessions.

Recording

Use tools/capture_helper.sh with --dir recordings around a Miaou command to record keystroke and frame JSONL artifacts. The helper prints the generated paths before launching the application.

Replay

Use tools/replay_tui.py to replay keystroke captures through a pseudo-TTY. Use tools/replay_screencast.sh to replay frame captures directly in a terminal.

Publishing

Use tools/convert_cast_to_gif.sh when an asciinema cast should become a GIF for documentation or release notes. Keep recordings short, focused, and easy to inspect.

Website media targets

Good showcase targets include gallery navigation, responsive resizing, theme switching, chart rendering modes, web viewer sessions, Miaou Force, Miaou Crypt, Miaou Links, and Geo Quiz.

Capture quality

Prefer recordings that show a complete interaction: navigation, a state change, visual feedback, and a clean ending. Link to source code beside media so readers can inspect the same flow.
