Capabilities

Side effects stay injectable.

Miaou applications can keep page and widget logic testable by routing system behavior through capability interfaces instead of direct global calls.

Registry

Capability.create, registration, lookup, requirement checks, and missing-capability reporting.

System

Filesystem access, environment variables, subprocess execution, disk usage, and writable-path probes.

Network

An injectable HTTP fetch capability for code that should remain replaceable in tests and headless runs.

Logger

Application-provided logging with enable/disable controls and logfile routing.

Clock and Timer

Cached driver-tick time, elapsed time, page-scoped intervals, and timeouts that are cleared on navigation.

Clipboard and Services

OSC 52 clipboard behavior, availability checks, and service lifecycle operations such as start, stop, restart, enable, and disable.

Palette

Injectable semantic colors and gradients for apps that need a central color policy.

API Reference

Open generated module documentation for exact signatures and types.
