Drivers

One UI, multiple surfaces.

Matrix

The default terminal backend uses cell-based double buffering and diff rendering. It is designed for responsive TUIs with frequent updates.

Lambda-Term

The compatibility backend builds on Lambda-Term and remains useful where terminal behavior is conservative or existing integrations depend on it.

SDL

The SDL backend unlocks pixel-perfect charts, images, and graphical rendering while keeping the same page/widget architecture.

Web

The web driver serves xterm.js over HTTP/WebSocket. The first connection controls the TUI; viewer connections can observe the same frames in real time.

Selection

Use MIAOU_DRIVER=matrix, term, sdl, or web to force a backend during development and demos.
