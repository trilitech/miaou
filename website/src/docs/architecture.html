Concepts

State, view, handlers.

Miaou applications are built from pages. A page owns immutable state, renders a view from that state, and handles keys by returning a new state.

Pages

For most new code, start with Direct_page: provide init, view, and on_key. Advanced pages can implement the full page signature when they need service cycles, custom modal handling, or explicit navigation state.

Widgets

Widgets are pure building blocks for input, display, layout, navigation, and feedback. They usually expose creation, rendering, key handling, and accessor functions.

Modals and prompts

The modal manager maintains a stack of overlays. Prompt helpers cover common confirm, input, and select flows without duplicating modal boilerplate.

Capabilities

Capabilities isolate side effects such as filesystem access, logging, services, networking, and clocks. This keeps application logic testable and replaceable.

Drivers

Drivers render ANSI frames or graphical output. Matrix is the default high-performance terminal driver; Lambda-Term is compatibility-oriented; SDL enables native graphics; Web exposes xterm.js sessions in a browser.
