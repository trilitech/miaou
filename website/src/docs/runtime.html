Runtime

Pages, keys, prompts, workflows.

The runtime layer connects page modules to navigation, key dispatch, modal prompts, user keymap overrides, and deterministic test workflows.

Registry

Register pages eagerly or lazily, override entries for integration points, and inspect key conflict reports before shipping a composed application.

User keymaps

Apps can support user overrides through MIAOU_KEYMAP_FILE or ~/.config/miaou/keymap.conf, keeping bindings configurable without editing page code.

Prompts and modals

Prompt helpers cover confirm, input, and select flows. The file browser modal provides a ready-to-use path picker for filesystem workflows.

Propagation

Handlers can consume an event or let it bubble, which keeps nested widgets, modals, and page-level shortcuts predictable.

Workflows

The typed workflow DSL lets tests drive deterministic headless UI flows and assert state transitions without requiring a live terminal.
