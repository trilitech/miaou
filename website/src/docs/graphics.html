Graphics

Terminal pixels and dense rendering.

Miaou includes terminal graphics paths for dashboards, games, maps, charts, and image-like output without leaving terminal cells.

Pixel modes

MIAOU_PIXEL_MODE can force or tune terminal pixel rendering. The framebuffer path can choose Sixel, Octant, Sextant, Half-block, or Braille depending on capability and desired fidelity.

Canvas

Octant and Braille canvas helpers provide high-density drawing for charts, maps, and framebuffer demos.

Globe

The globe widget supports coastline projection and haversine helpers for location-oriented terminal interfaces.

Detection

Terminal capability detection decides what can be drawn safely and lets applications degrade to simpler output when required.
