Theme by meaning.

Widgets should use semantic style helpers such as themed_success, themed_warning, themed_error, themed_muted, and themed_emphasis. Themes decide the actual colors.

Contextual styles

The snippet renders the status line shown above by choosing themed_success, themed_warning, or themed_error from a typed status value. Container widgets can provide contextual selectors so themes can style rows, focused children, selected items, and alternating layouts without changing widget code.

Raw colors

Use raw ANSI colors only for charts, gradients, and pixel graphics where precise color ramps are part of the visual output.
