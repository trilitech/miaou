Testing

Verify terminal behavior without guessing.

Miaou supports headless execution, deterministic workflows, capture/replay artifacts, and runtime verification patterns for terminal applications.

Headless

Use the headless driver and typed workflow DSL to run key sequences, inspect state, and assert rendered output without opening a terminal.

Capture internals

Keystroke and frame JSONL hooks let tests replay user input and inspect frame output. Cast conversion turns real sessions into website media.

Terminal runners

Term test runners and tmux/Playwright-style verification help confirm behavior that depends on actual terminal dimensions, focus, or driver output.

Web viewer

The standalone web viewer can expose controller and observer sessions, useful for demos, visual checks, and debugging remote terminal flows.
