Documentation

Build with Miaou confidently.

These docs are organized around the way Miaou applications are built: pages own state, widgets render reusable UI, capabilities isolate side effects, and drivers decide where frames are displayed.

Start with the course when you want a guided path, or jump into architecture, drivers, styling, and capture tooling when you need a specific topic.

Fastest path

Install the library, run a demo, then open the first lesson when you are ready to wire your own page.

opam install miaou dune exec -- miaou.demo MIAOU_DRIVER=web dune exec -- miaou.demo
