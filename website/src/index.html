OCaml terminal UI library

Build fast terminal apps in OCaml.

Miaou gives OCaml applications a state-driven page model, reusable widgets, responsive layouts, semantic theming, multiple rendering backends, and capture tooling for tests and documentation.

Quick start

Install once, choose the driver at runtime.

opam install miaou dune exec -- miaou.demo MIAOU_DRIVER=web dune exec -- miaou.demo

Framework

A small architecture for real terminal products.

SVH

State-view-handlers

Pages keep immutable state, render from state, and return updated state from key handlers.

DP

Direct_page

Start with only init, view, and on_key, then opt into advanced page lifecycle hooks.

UI

Rich widgets

Tables, forms, text editors, charts, file browsers, QR codes, images, trees, layouts, and more.

DRV

Multiple drivers

Matrix, Lambda-Term, SDL, and web/xterm.js backends cover terminal, graphical, and browser use.

CSS

Themeable by design

Semantic styles and contextual selectors let applications adapt across dark, light, and high-contrast themes.

REC

Test and record

Headless runs, frame captures, replay scripts, asciicasts, and GIF conversion help you test and share terminal flows.

Follow the course in order to build a page, adapt it to terminal size, add charts, apply theme-safe styling, and record the result.
