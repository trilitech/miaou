Course 3

Render compact metrics.

Sparklines are useful for dashboards, status panes, and service monitors. This lesson extends the responsive dashboard with compact telemetry.

You will learn: sparkline creation, bounded sample history, Braille rendering, and threshold-based color payloads.

Color payloads

The snippet builds the sparkline shown above by pushing bounded latency samples, rendering them with the label Latency, and applying a red threshold once values cross 120ms.

Chart color fields expect ANSI SGR foreground payloads, such as 38;5;81, not raw palette indices. Threshold colors override the default render color for matching values.

Rendering modes

Use ASCII mode for maximum compatibility, Braille mode for smoother curves, and Octant mode when the terminal supports Unicode 16 octant characters.
