Course 1

Your first Miaou page.

This lesson introduces the core Miaou loop: initialize state, render from state, react to keys, and request navigation.

You will learn: page state, pure rendering, key handling, and the Direct_page.quit navigation effect.

What the page does

init creates the initial count, view renders the same boxed counter shown above, and on_key returns updated state for Up and Down. Calling Direct_page.quit () raises a navigation effect that the page wrapper translates into a quit request.

Run pattern

In a real executable, initialize the Eio runtime, register capabilities your app needs, wrap the module as a Miaou page, and pass it to the runner. The full demo gallery contains many examples of that wiring.

Next steps

Add widgets, split larger views into helper functions, and keep expensive computation outside view so rendering stays fast. Then continue to Course 2 to make layouts react to terminal width.
